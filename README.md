# iOSStudy

## GalleryApp
URL에서 이미지를 불러와 보여주는 사진첩 앱  
- CollectionView Customized Layout 적용 -> 컬럼 수 변경기능 추가  
- networking 비동기 처리 적용 -> 부드러운 UI 스크롤 구현  
- Dictionary를 사용한 이미지캐싱 적용 -> 이미지 다운로드 반복 방지
